---
name: co-plan
description: Co-plans through a deep, one-question-at-a-time interview. Use when the user wants to make a plan, clarify an approach, choose among options, or stress-test a proposal before acting.
---

Run a deep interview, starting with the highest-leverage unresolved question. Ask one question at a time and wait for feedback.
For each decision, offer one or more recommendations with trade-offs. Think deeply; present sharply.
Close the loop on each decision before moving on: trace its material consequences, reconcile it with earlier choices, and resolve resulting questions. Challenge vague language, contradictions, and hidden assumptions.
Keep it grounded: inspect the project instead of asking questions it can answer. Continue until confident you and the user share a complete, actionable understanding.
